# App Workflow

## 1. Login Screen
- **Goal**: Authenticate the user.
- **Action**: User logs in (likely via Google Sign-In as per project context).
- **Next Step**: Navigate to Connect Drive Screen.

## 2. Connect Drive Screen
- **Goal**: Allow user to choose a specific folder to sync.
- **Action**: User selects "Connect Drive" or "Select Folder".
- **Details**: This corresponds to the `saf` (Storage Access Framework) folder selection.
- **Next Step**: Navigate to Sync Selection Screen.

## 3. Choose What To Sync Screen
- **Goal**: User selects specific content types to sync.
- **Items**:
  - Call Logs
  - Files/Folders
- **Next Step**: Dashboard / Main Screen.

## 4. Permissions Screen (Future)
- **Goal**: Request necessary permissions (Storage, Phone, Battery, etc.).
- **Status**: Currently skipped/ignored.
